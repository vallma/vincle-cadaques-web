# Assets Necesarios para la Web

## Videos (para efectos Parallax)

El usuario proporcionará estos. Listar aquí lo que idealmente se necesita:

### Videos prioritarios
- [ ] **Hero video** — Vista general del restaurante o Cadaqués. Ideal: aéreo o toma lenta del pueblo y el mar
- [ ] **Interior** — Comedor abovedado con luz cálida, paredes de piedra, mesas de madera
- [ ] **Cocina** — Platos siendo preparados, ingredientes frescos, texturas
- [ ] **Exterior / Terraza** — Terraza exterior, callejón del casco antiguo
- [ ] **Mar** — El mar Mediterráneo de Cadaqués, luz de atardecer, rocas
- [ ] **Detalles** — Close-ups de platos, manos cocinando, productos del mar

### Formato recomendado para parallax
- Resolución: 1920x1080 mínimo, 4K ideal
- Formato: MP4 (H.264 o H.265)
- Duración: loops de 10-30 segundos
- Sin audio necesario (se puede usar, pero opcional)
- Framerate: 24fps o 30fps

---

## Imágenes

- [ ] **Logo** — Si existe, en PNG con fondo transparente y en blanco
- [ ] **Fotografía de platos** — Mínimo 6-8 platos del menú
- [ ] **Fotografía del espacio** — Interior y exterior, distintas horas del día
- [ ] **Fotografía del equipo** — Opcional, humaniza el restaurante
- [ ] **Fotografía de Cadaqués** — Pueblo, casco antiguo, mar (puede ser stock)

### Formatos de imagen
- JPG: fotografías
- PNG: logos, iconos con transparencia
- WebP: versiones optimizadas para web

---

## Notas

- Las imágenes/videos del usuario se deben colocar en `/assets/images/` y `/assets/videos/`
- Nombrar archivos descriptivamente: `hero-interior.mp4`, `plato-croquetes-gamba.jpg`, etc.
- Se necesitará optimización para web (compresión sin pérdida de calidad visible)
